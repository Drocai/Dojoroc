# Roc sprites — drop your art here

Replace any Roc's built-in SVG with your own art in **one step**: drop a
transparent PNG named after the species key into this folder. No code changes.

| File to add            | Roc                |
|------------------------|--------------------|
| `zen-pebble.png`       | Zen Pebble         |
| `makiwara-rock.png`    | Makiwara Rock      |
| `nunchuck-rocks.png`   | Nunchuck Rocks     |
| `bo-staffali.png`      | Bo Staffali        |
| `smash-boulder.png`    | Smash Boulder      |
| `sumo-bava.png`        | Sumo Bava          |
| `ninja-obsidian.png`   | Ninja Obsidian     |
| `guardian-gargoyle.png`| Guardian Gargoyle  |
| `pa-geode.png`         | Pa Geode           |
| `sensei-stone.png`     | Sensei Stone       |

**Tips**
- Square, transparent background (256×256 or larger). `.png`, `.webp`, or `.svg` work.
- The image inherits idle-bob / poke / reaction animation automatically.
- Any species *without* a file keeps its themeable SVG, so you can swap them in one at a time.
